# AGENTS.md — маніфест для ШІ

Інструкції для ШІ-агентів (Cursor та ін.) при роботі з **concept-docs** — документацією для [Concept Core](https://github.com/php-concept/core) (`php-concept/concept-core`)
та [Concept Skeleton](https://github.com/php-concept/concept-skeleton) (`php-concept/concept-skeleton`)

## Про проєкт

Concept Skeleton - мінімальний додаток для старту розробки додатків щоб не конфігурувати і не налаштовувати все з нуля.
Concept Skeleton базується на використанні concept-core

Для побудови документації код обох проєктів знаходиться в папках code/core, code/skeleton.

Приклади компонентів у документації (AuthAdmin, DebugBar тощо) базуються на реальному коді з папки code. **Не згадуйте skeleton-dev** у тексті документації.

## Що робити
Документація ведеться двома мовами: українською та англійською.
При формуванні документації необхідно переглядати папку code для повної відповідності до поточної реалізації.
Крім того потрібно ознайомитися README.md

## Що НЕ робити

- Не робити git commit без явного запиту користувача
- Не змінювати нічого в пвпці code

## Мова спілкування

Користувач спілкується українською. Відповіді та коментарі в коді — англійською (як у наявному коді), якщо користувач не просить інакше.
