Вступ

Concept Core — це ядро PHP-фреймворку Concept. Воно забезпечує bootstrap застосунку, контейнер сервісів, HTTP-шар, маршрутизацію, представлення, валідацію та підключаємі компоненти.

Ця документація описує пакет php-concept/core — фундамент для skeleton-проєктів та власних рішень.

Вперше з Concept?

Рекомендуємо почати з Встановлення, потім Конфігурація та Життєвий цикл запиту.

Чому Concept?

Concept створений для розробників, яким потрібен сучасний PHP-стек без зайвої ваги моноліту:

  • League Container — впровадження залежностей
  • League Route — HTTP-маршрутизація
  • Twig — шаблони
  • Illuminate Database — моделі в стилі Eloquent
  • Valinor — мапінг даних запиту в DTO
