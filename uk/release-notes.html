Concept Core 1.x

Перша стабільна лінійка engine з акцентом на PSR і модульність:

  • PSR-7 HTTP, PSR-11 DI (League Container + Reflection), PSR-15 middleware
  • League Route, RouteStrategy (FormRequest, route params, interceptors)
  • Illuminate Database (Eloquent, міграції, pagination)
  • Rakit validation, Valinor DTO casting, Twig views
  • Компонентна реєстрація: routes, migrations, seeders, views, CLI
  • CSRF, data masker, safe redirects, telemetry hooks
  • Symfony Console, Whoops / production fallbacks

Concept Skeleton

Мінімальний застосунок для старту: bootstrap/, config/, routes/web.php, один контролер і frontend-шаблон.

Документація 1.x

Статичний сайт (Bootstrap 5, Highlight.js): розділи від встановлення до компонентів, українська та англійська версії. Зміст звіряється з кодом у code/core і code/skeleton.
