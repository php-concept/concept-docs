Знайомство з Concept

Найкращий спосіб почати — skeleton-застосунок, який уже містить php-concept/core та робочий bootstrap.

Вимоги до сервера

Concept Core потребує:

  • PHP 8.4 або новіший
  • Composer 2.x
  • Розширення: mbstring, json, pdo

Встановлення через Composer

composer require php-concept/core:^1.0

Bootstrap застосунку

Створіть екземпляр застосунку у front controller (наприклад, public/index.php):

<?php declare(strict_types=1);

use Concept\Core\App;

$app = App::create(
    rootPath: dirname(__DIR__),
    paths: [
        'config' => 'config',
        'storage' => 'storage',
        'routes' => 'routes',
    ],
);

$app->registerServiceProviders([
    $app->getRootPath() . '/bootstrap/providers/app.php',
]);

$app->run();
