Introduction

Concept Core is the kernel of the PHP Concept framework. It provides the application bootstrap, service container, HTTP layer, routing, views, validation, and pluggable components that applications build upon.

This documentation covers the php-concept/core package — the foundation used by skeleton applications and custom projects. For application-level topics (components, modules, deployment), refer to your project documentation.

New to Concept?

We recommend starting with Installation, then Configuration and Request Lifecycle.

Why Concept?

Concept is designed for developers who want a modern PHP stack without the weight of a full monolith. The core is explicit, composable, and PSR-friendly:

  • League Container for dependency injection
  • League Route for HTTP routing
  • Twig for templating
  • Illuminate Database when you need Eloquent-style models
  • Valinor for mapping request data to DTOs
