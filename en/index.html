Introduction

Concept Core is the kernel of the PHP Concept framework. It provides application bootstrap, service container, HTTP layer, routing, views, validation, and pluggable components.

This documentation describes the php-concept/core package — the foundation for your own solutions.

New to Concept?

We recommend starting with Installation, then Configuration and Request Lifecycle.

Why Concept?

Concept is built for developers who need a modern PHP stack without the weight of a monolith:

  • League Container — dependency injection
  • League Route — HTTP routing
  • Twig — templating
  • Illuminate Database — Eloquent-style models
  • Valinor — mapping request data to DTOs
