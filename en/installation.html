Installation

Meet Concept

The recommended way to start a new project is to use the official skeleton application, which already includes php-concept/core and a working bootstrap.

Server Requirements

Concept Core requires:

  • PHP 8.4 or higher
  • Composer 2.x
  • Extensions: mbstring, json, pdo (when using the database component)

Installing via Composer

composer require php-concept/core:^1.0

Bootstrapping the Application

Create the application instance in your front controller (e.g. public/index.php):

<?php declare(strict_types=1);

use Concept\Core\App;

$app = App::create(
    rootPath: dirname(__DIR__),
    paths: [
        'config' => 'config',
        'storage' => 'storage',
        'routes' => 'routes',
    ],
);

$app->registerServiceProviders([
    $app->getRootPath() . '/bootstrap/providers/app.php',
]);

$app->run();
